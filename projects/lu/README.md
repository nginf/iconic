# @nginf/iconic-lu

this package is part of [@nginf/iconic](https://iconic.nginf.dev) library. It provides Lucide icons

## install
`npm install @nginf/iconic-lu`

# use
```ts
   // ts
   import { LuChevronDown } from '@nginf/iconic-lu';   
   ...
   // html
  <lu-chevron-down></lu-chevron-down>
 ```
